## Sobre o Projeto

Um sistema prático para o monitoramento e gestão de portfólio temporariamente feito apenas para cripto moedas.
O projeto permite o acompanhamento de saldos, registro de preços de compra e o cálculo de lucros ou perdas em tempo real, fornecendo uma visão clara do desempenho dos ativos.
Os dados de mercado são alimentados através da integração direta com a API da CoinMarketCap.
